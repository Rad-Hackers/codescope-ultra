# CodeScope Ultra

منصة تحليل مواقع احترافية بواجهة عربية افتراضية واتجاه RTL بالكامل، مع زر تبديل إلى الإنجليزية.

## ما الذي تم تجهيزه
- العربية هي اللغة الافتراضية عند فتح الموقع أول مرة.
- لا توجد روابط قناة، ولا اسم مطور، ولا سنة إصدار، ولا حقوق خارجية داخل الواجهة.
- Node.js Backend حقيقي لجلب Source Code بدل الاعتماد على المتصفح و CORS.
- واجهة فخمة Dark/Light، متجاوبة مع الهاتف، وجاهزة للنشر على Render.

## الميزات الحقيقية
- جلب Source Code عبر Backend.
- تحليل SEO و Meta Tags و OpenGraph و Structured Data.
- اكتشاف CMS و Frameworks و CDN وأدوات التحليلات والدفع.
- فحص Security Headers و Cookies.
- تحليل جودة الكود بقواعد حقيقية، وليس AI وهمي.
- استخراج الروابط والصور والسكريبتات وملفات CSS والدومينات الخارجية.
- اكتشاف روابط API المحتملة داخل الكود.
- DNS Lookup و WHOIS.
- قراءة robots.txt و sitemap.xml.
- Screenshot حقيقي عند توفر Chromium في الاستضافة.
- مقارنة كودين Diff.
- تحليل HTML محلي من ملف أو لصق كود.
- تصدير PDF / JSON / CSV / HTML.
- سجل محلي لآخر 100 تقرير.

## التشغيل المحلي
```bash
npm install
npm start
```
ثم افتح:
```text
http://localhost:3000
```

## النشر على Render
ارفع المشروع إلى GitHub ثم أنشئ Web Service في Render. الملف `render.yaml` جاهز.

> ملاحظة صادقة: بعض المواقع تمنع الجلب أو Screenshot بسبب Cloudflare أو سياسات حماية أخرى. في هذه الحالة سيعرض الموقع الخطأ الحقيقي بدل عرض نتيجة وهمية.
